# Documentation

Add your demo GIFs and screenshots here:

- `demo-detection.gif` - Screen recording of auto-detection in action
- `demo-analytics.gif` - Screen recording of analytics/counting feature
- `screenshot-main.png` - Main interface screenshot

## How to Create Demo GIFs

1. Use a screen recorder like:
   - [ScreenToGif](https://www.screentogif.com/) (Windows)
   - [Peek](https://github.com/phw/peek) (Linux)
   - [Kap](https://getkap.co/) (Mac)

2. Record the application in action
3. Save as GIF and place in this folder
4. Update README.md paths if needed
